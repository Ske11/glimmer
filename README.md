# 浮光 Glimmer

一款 macOS 桌面浮动组件应用，每个组件都是独立的透明无边框窗口，叠在桌面壁纸之上，提供沉浸式的玻璃质感体验。

## 功能

**5 种组件**
- 时钟 — 当前时间、日期与问候语
- 待办 — 轻量待办清单
- 心语 — 自定义语录展示
- 倒计时 — 目标日期倒数
- 便签 — 自由文字记录

**8 种粒子场景主题**
- 雨天 / 雪景 / 暮色 / 星夜 / 极光 / 花间 / 薄雾 / 流萤

每种主题带有独立的粒子动效和场景配色，组件背景、边框、气泡均随主题变化。

**交互特性**
- 拖拽移动 — 鼠标按住任意位置即可拖动
- 自由缩放 — 右下角拖拽调整大小
- 收起/展开 — 收起为小圆点气泡，点击恢复
- 常驻桌面 — 组件固定在所有工作区，切换 Space 不会消失
- 系统托盘 — 通过菜单栏图标快速新建组件、显示全部或退出

## 技术栈

- **Tauri 2** — 原生窗口管理，多窗口架构
- **React + TypeScript** — 前端 UI
- **Vite** — 构建工具
- **Canvas 2D** — 粒子动效渲染
- **objc2 / AppKit** — macOS 原生 API（窗口行为控制）

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run tauri dev

# 构建
npm run tauri build
```
